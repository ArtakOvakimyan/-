To get the result of your search - type in cmd console the next string:
"node [name of script-file] [name of txt file with main text] [name of txt file with string] [Method you want to use]"
Example: node search.js text.txt string.txt RabinKarp
Names of methods to use: "BruteForce", "SimpleHash", "SquareHash", "RabinKarp"
